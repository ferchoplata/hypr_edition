# Roadmap

## Version 0.1

- Instalador post-instalacion funcional para CachyOS.
- Deteccion inicial de Arch Linux y CachyOS.
- Instalacion de paquetes base de Hyprland.
- Backup automatico de configuraciones existentes.
- Dotfiles iniciales para Hyprland, Waybar, Kitty y Rofi.

## Version 0.2

- Mejorar el menu actual sin romper la instalacion base.
- Separar perfiles: minimo, completo y gamer.
- Agregar soporte de AUR con deteccion de helper instalado.
- Agregar exportacion del sistema actual.

## Version 0.3

- Detectar GPU para instalar drivers graficos correctos sin adivinar.
- Agregar restauracion de backups.
- Convertir partes del instalador en modulos reales.
- Agregar desinstalacion segura de componentes opcionales.

## Version 0.4

- Agregar temas y wallpapers propios.
- Preparar pruebas en maquina virtual.
- Evaluar una experiencia mas cercana a HyDE sin abandonar Arch/CachyOS como base.
